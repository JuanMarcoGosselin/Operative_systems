# CPU Scheduler Simulator (homework3)

This repository contains a CPU scheduler simulator with multiple algorithms (FIFO, SJF, STCF, Round Robin, MLFQ) and a GTK GUI skeleton.

## Build

Requirements: gcc, pkg-config, GTK+-3.0 development headers.

bash
make
# run GUI (binary calls gui_gtk_main internally in scheduler.c)
./build/bin/scheduler

See top of this document for file listing. Each section above contains the initial skeleton of a file; copy each section into separate files under the same paths for a working tree.
